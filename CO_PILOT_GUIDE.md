## Quick Start

### Your First Co-Pilot Session (60 seconds)

1. **Open the Extension**
   - Click the LinkedIn Lead Finder icon in your Chrome toolbar
   - You'll see the **Co-Pilot** tab (it's the first tab)

2. **Analyze Your Current Page**
   - Click the **üîç Analyze Page** button
   - The co-pilot will detect what type of LinkedIn page you're on
   - You'll see what can be extracted (posts, profiles, comments, etc.)

3. **Select Your Goal**
   - Click on one of the 6 goal cards
   - Each goal is designed for a specific type of extraction
   - The card will highlight when selected

4. **Start Extraction**
   - Click **üöÄ Start Co-Pilot**
   - Watch the status updates in real-time
   - Leads will automatically save to your Google Sheets

That's it! The co-pilot is now working for you.

### LinkedIn-Specific Tips

**1. LinkedIn Rate Limits**
- LinkedIn may slow you down if you're too aggressive
- Use reasonable scroll cycles (default: 6)
- Add delays between actions (default: 1000ms)
- Take breaks between long sessions

**2. Better Search Results**
- Use LinkedIn's filters before starting co-pilot
- More specific search = better quality leads
- Example: Filter by location, company size, seniority

**3. Best Times to Use**
- Early morning or evening (less LinkedIn activity)
- LinkedIn's servers are less loaded
- Extraction is faster

**4. Profile Completeness**
- More complete profiles have more extractable data
- Premium profiles often show more contact info
- Some data requires being connected

## Troubleshooting

### Co-Pilot Won't Start

**Problem:** Clicking "Start Co-Pilot" does nothing

**Solutions:**
1. Make sure you selected a goal (card should be highlighted)
2. Check that you're on a compatible LinkedIn page
3. Try analyzing the page first
4. Refresh LinkedIn and try again
5. Check browser console for errors (F12)

---

### No Data Being Extracted

**Problem:** Co-pilot runs but no leads appear in Google Sheets

**Solutions:**
1. Check Google Sheets tab - are you connected?
2. Verify Sheet ID is correct
3. Test connection in Google Sheets tab
4. Check Statistics tab - are leads being found?
5. Try disabling AI filtering temporarily
6. Check that auto-sync is enabled

---

### Wrong Goal for Page Type

**Problem:** Selected goal doesn't work on current page

**Solution:**
1. Click "Analyze Page" to see compatible goals
2. Choose a goal that matches the page type
3. Page type detection:
   - Job pages ‚Üí Job Applicants
   - Post pages ‚Üí Comment Mining, Post Engagement
   - Search results ‚Üí People Discovery, Keyword Hunting
   - Company pages ‚Üí Company Intel

---

### Extraction Stopped Early

**Problem:** Co-pilot stops before finishing

**Solutions:**
1. Check if you hit LinkedIn's rate limit (refresh and try later)
2. Check if page structure changed (LinkedIn updates frequently)
3. Look for error messages in status
4. Try with autopilot disabled
5. Reduce scroll cycles in settings

---

### Autopilot Not Working

**Problem:** Autopilot checkbox enabled but extraction doesn't continue on new pages

**Solutions:**
1. Make sure co-pilot is started (not just checkbox enabled)
2. New page must be compatible with selected goal
3. Try navigating to a different compatible page
4. Stop and restart co-pilot
5. Check that "Autopilot enabled" shows in status

---

### LinkedIn Blocking/Slowness

**Problem:** LinkedIn showing "Please slow down" or similar

**Solutions:**
1. Stop co-pilot immediately
2. Wait 15-30 minutes
3. Reduce scroll cycles in Settings tab
4. Increase delays between actions
5. Use co-pilot in shorter sessions

---

### Browser Performance Issues

**Problem:** Browser becomes slow during extraction

**Solutions:**
1. Close other tabs
2. Disable browser extensions temporarily
3. Reduce scroll cycles
4. Extract in smaller batches
5. Use incognito mode (fewer extensions)

---

### Data Not Saving to Sheets

**Problem:** Leads found but not in Google Sheet

**Solutions:**
1. Check Statistics tab ‚Üí "Exported" count
2. If auto-sync disabled, click "Export Now"
3. Verify Google account is connected
4. Test Sheet connection
5. Check Sheet permissions (must have edit access)
6. Try re-authenticating Google account

---

### Missing Contact Information

**Problem:** Leads extracted but no emails/phones

**Solutions:**
1. Not all LinkedIn profiles show contact info publicly
2. Try Comment Mining goal (more contact info in comments)
3. Some info only visible to connections
4. Enable "scan comments" in Settings
5. Use AI to enrich data later

---

### Custom Instructions Not Working

**Problem:** AI seems to ignore custom instructions

**Solutions:**
1. Make sure AI features are enabled in Automation & AI tab
2. Add OpenRouter API key if using AI filtering
3. Be very specific in instructions
4. Try simpler, clearer language
5. Check that OpenRouter model is selected
